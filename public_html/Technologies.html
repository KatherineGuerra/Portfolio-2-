Technology

HTML

Html stands for Hypertext Markup Language. This language outlines the for the formation of online pages. HTML gives the designer a way to show online information documents with tables, headings, texts and photos. It also helps to design forms to make transactions with remote services, for example making reservations and ordering products. It also helps the designer to attach into the created documents different features such as video clips, and sound clips.

CSS

CSS stands for "Cascading Style Sheets". This is the language for outlining the presentation of Web pages. This includes colors, fonts, and backgrounds. It helps designers to adapt the presentation to either large or small screens. CSS helps with the visual way in which text or pictures are set out on a page.

BOOTSTRAP

Bootstrap is a collection of tools to make websites and web applications. Bootstrap contains HTML and CSS based design templates. It makes front-ent web build up faster and easier. Bootstrap was made for people of all skill levels, devices of all shapes, and projects of all sizes.

JavaScript

JavaScript is a computer programming language that stores in web pages in programs. The JavaSript code tells you where are your errors if your website is not working well.
